simple-meanshift-traker

## 概要
Mean-shift Trackingの簡単な実装例です

## 特徴
- 物体のHue成分をもとに追跡
- 追跡窓サイズは固定

## Requirement
OpenCVが必要です

## ビルド
```
$ git clone https://github.com/gishi523/simple-meanshift-traker.git
$ cd simple-meanshift-traker
$ mkdir build
$ cd build
$ cmake ../
$ make
```

## 実行
./simple-meanshift-traker [input movie]
